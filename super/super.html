    OR(||):

    The logical OR ( || ) operator also takes two expressions: expression1 || expression2 ; NB: It returns the first expression that is true, either or the left or the right. There are three different ways the || operator can be evaluated:

    Left side Right side Return value Truthiness or return Value
    Truthy Doesn't matter Left side Truthy
    Falsey Truthy Right side Truthy
    Falsey Falsey Right side Falsey
    NOTE:: 1. If the left-side expression is truthy, the right-side expression doesn't matter at all. The || operator returns the left side's truthy value and completes. 2. If the left-side expression is falsey, the || operator returns the right side's value (regardless of whether it's truthy or falsey) and completes
